# 股票推荐 Skills 配置

通过自然语言定义股票筛选和推荐偏好，系统在每日推荐时会根据以下规则自动筛选股票。

## 自定义筛选规则

在此文件中用简体中文添加筛选偏好，每行一条规则。

### 当前生效规则
- 筛选近30天成交量温和放大且市盈率低于行业平均15%的A股个股
- 筛选的股票当前价格需要在50元以内
- 排除ST以及有退市风险标的
- 核查筛选出来的个股连续三年净利润增速、ROE水平逐步升高的
- 排除净利润连续下滑、有财务暴雷风险的
- 核查这些个股的北向资金与主力资金近10天的流入情况，保留资金逐渐加仓的
- 评估这些筛选过的个股，保留未来三年市场空间充足，政策倾斜明显的
